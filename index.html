¿Cómo se juega?

El objetivo es acumular palabras y dinero para dominar la acentuación.
Ganas al conseguir la palabra Sobreesdrújula.

  1. Objetivo: Compra palabras para ganar dinero pasivo. Reúne 30 de cada categoría para desbloquear y comprar la palabra Sobreesdrújula y ganar.
  2. El Desafío: Ocasionalmente, se te pedirá escribir una palabra de una categoría en 30s. ¡Cuidado! 3 fallos reinician tu progreso.
  3. Bonus: Busca palabras Super Value (rojas) para potenciar una categoría, o el Bonus CHRISTOPHER para potenciar todo.
  4. Limbo Gramatical: ¡Mantente alerta a eventos raros! El Limbo Gramatical: ??? es una fisura en la realidad que te ofrece grandes recompensas.
  5. El Ladrón de Tildes: ¡Ten cuidado! Cuando el Ladrón se acerca, susurra. Durante este tiempo, podría intentar engañarte con palabras corruptas: ???. Comprarlas tendrá consecuencias.
  6. Tienda y Prestigio: Usa la Tienda para comprar Vidas y Escudos. Al ganar y Formatear, obtienes un multiplicador permanente y Puntos de Prestigio, que puedes usar para mejoras haciendo clic en el botón del multiplicador.
