¿Cómo se juega?

El objetivo es acumular palabras y dinero para dominar la acentuación.
Ganas al conseguir la palabra Sobreesdrújula.

  1. Objetivo: Compra palabras para ganar dinero pasivo. Reúne 30 de cada categoría para desbloquear y comprar la palabra Sobreesdrújula y ganar.
  2. El Desafío: Ocasionalmente, se te pedirá escribir una palabra de una categoría en 30s. El juego verificará tu palabra en línea para asegurar total precisión. ¡Cuidado! 3 fallos reinician tu progreso.
  3. Bonus: Busca palabras Super Value (rojas) para potenciar una categoría, o el Bonus CHRISTOPHER para potenciar todo.
  4. Limbo Gramatical: ¡Mantente alerta a eventos raros! El Limbo Gramatical: ??? es una fisura en la realidad que te ofrece grandes recompensas. Durante el evento, Christopher podría intentar comunicarse contigo...
  5. El Ladrón de Tildes: ¡Ten cuidado! Cuando el Ladrón se acerca, susurra. Podría intentar engañarte con palabras corruptas: ???, e incluso se burlará de tus fallos en los retos.
  6. Tienda y Prestigio: Usa la Tienda para comprar Vidas y Escudos. Al ganar y Formatear, obtienes un multiplicador permanente y Puntos de Prestigio, que puedes usar para mejoras permanentes, ¡incluyendo algunas para contrarrestar al Ladrón!
